# Web-Development
Nơi lưu trữ những bài tập của môn "Phát triển ứng dụng Web" của K23
- Kiểm tra máy đã có node.js và npm chưa.
(Sử dụng cú pháp node -v với npm -v trên window terminal kiểm tra).
- Sau khi đã xác nhận có rồi, sử dụng câu lệnh sau:
    + npm init -y (dòng này sẽ sinh ra file package.json).
        Tại đó, thêm "types" : module, ở đầu dòng.
    + npm install express (sinh ra node-modules folder và package-lock.json).
- Folder views là những file html nhằm hiện nội dung.
- Folder publics là những file tĩnh như ảnh, file css.
- Tạo file app.js để code web server.
- Code xong dùng câu lệnh node app.js để kiểm tra.
